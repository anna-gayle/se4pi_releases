# SE4PI Audio Client Releases

This repository contains release files for the SE4PI Audio Client.

## Installation Instructions

1. **Download the latest release** from the Releases section
2. **Extract the zip file** to a folder of your choice
3. **Install VB-Cable** for virtual audio routing
4. **Install RNDIS drivers** for USB connection
5. **Run SE4PI-Client.exe**

**Note:** The server runs on a Raspberry Pi with SE4PI software installed.
